Module resampWorkerService

Description

メインスレッド側から Worker をロードし、ResampWorkerRequest に変換したデータを送信して、 Worker 側の音声合成処理結果（Float64Array）を受け取るサービスです。

このサービスは、React UI から利用され、VoiceBank (vb) データから wav や frq データを取得し、必要な変換を実施した上で、Worker に転送可能な形（Float64Array）で送信します。
